  <name>Task 2: Create sidecar download script and capabilities config</name>
  <files>

  <read_first>

    src-tauri/tauri.conf.json (verify externalBin paths from Task 1)
  </read_first>
  <action>
    1. Create `scripts/download-sidecars.sh` (chmod +x) that:
       a. Detects the host target triple via `rustc --print host-tuple` (Rust 1.84+), falling back to `rustc -Vv | grep host | cut -f2 -d' '` for older versions.
       b. Creates `src-tauri/binaries/` directory if it does not exist.
       c. Downloads yt-dlp for the detected platform:
          - `aarch64-apple-darwin` or `x86_64-apple-darwin`: download `yt-dlp_macos` from `https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp_macos`
          - `x86_64-pc-windows-msvc`: download `yt-dlp.exe` from `https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe`
          - `x86_64-unknown-linux-gnu`: download `yt-dlp` from `https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp`
       d. Renames the downloaded file to `yt-dlp-<TARGET_TRIPLE>[.exe]` in `src-tauri/binaries/`.
       e. Downloads ffmpeg static build for the detected platform:
          - macOS: from `https://evermeet.cx/ffmpeg/getrelease/zip` (extract binary from zip)
          - Windows: from `https://github.com/BtbN/FFmpeg-Builds/releases/latest/download/ffmpeg-master-latest-win64-gpl.zip` (extract ffmpeg.exe)
          - Linux: from `https://johnvansickle.com/ffmpeg/releases/ffmpeg-release-amd64-static.tar.xz` (extract ffmpeg binary)
       f. Renames ffmpeg binary to `ffmpeg-<TARGET_TRIPLE>[.exe]` in `src-tauri/binaries/`.
       g. On macOS/Linux: runs `chmod +x` on both binaries.
       h. Prints a summary: "Sidecars ready for <TRIPLE>: yt-dlp-<TRIPLE>, ffmpeg-<TRIPLE>"

    2. Create `src-tauri/capabilities/default.json`:
       ```json
       {
         "identifier": "default",
         "description": "Default capabilities for the main window",
         "windows": ["main"],
         "permissions": [
           "core:default",
           "shell:allow-spawn",
           {
             "identifier": "shell:allow-execute",
             "allow": [
               { "name": "binaries/yt-dlp", "sidecar": true },
               { "name": "binaries/ffmpeg", "sidecar": true }
             ]
           },
           "dialog:allow-open",
           "store:allow-load",
           "store:allow-set",
           "store:allow-get",
           "store:allow-save"
         ]
       }
       ```

    3. Add `src-tauri/binaries/` to `.gitignore` (sidecar binaries are large, downloaded at build time, not committed).

    4. Run the download script to populate binaries for the current dev machine (macOS ARM expected).
  </action>
